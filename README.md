#  Personal Finance & Spending Behaviour Analysis

### Project Overview
This project analyses personal finance and spending behaviour using a dataset of individual financial patterns. The analysis focuses on understanding how income, expenses, and saving behaviour vary across individuals.

---

### Objective
To identify key factors influencing saving behaviour and financial decision-making, and provide insights that support better personal financial management.

---

### Tools Used
- Excel (data cleaning and analysis)  
- Power BI (data visualization and dashboard development)  

---

### Dataset
Personal finance dataset (cleaned and prepared for analysis).

---

### Key Insights
- Income level has a strong influence on saving behaviour  
- Higher expenses are associated with lower savings  
- Spending patterns vary across different income groups  

---

### Output
Interactive dashboard developed to visualise financial patterns and behaviour trends.
